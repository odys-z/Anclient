Quick Start

  1. download volume assets here, extract date into the volume path set in WEB-INF/settings.json, e.g. ../vol;
  2. eidt dist/private/host.json, point jserv to web service in docker container. See github.json for example;
  3. load this page with a web server, e.g. 'python3 -m http.server 8888';
  4. start the synode with 'java -jar bin/jserv-album-0.7.0.jar, from where the released synode zip file unzipped.

See also AnTreeditor docfor quick start, and Example: doc type parser for customizing cells.
