Test AnTreeditor¶

Test page:

js/anreact/test/sessionless/dist/widgets.html

This front page work with jserv-sandbox. For @anclient/anreact v0.4.36, a docker image, jsandbox:treeditor, for testing AnTreeditor is published.

To install Docker, start the container with scripts:

./docker-start

This will pull a docker image, odysz/jsandbox:treeditor, into local container.
