About Anreact

Anreact, npm package @ancllient/anreact, is a collection of ReactJs UI components used for binding json data provided by @anclient/semantier, the semantic data layer at typescript client.

Running the Tests

The @anclient/anreact (typescript) is test with three types of test:

mocha tests
jsample
session-less widgets

To run mocha tests, in js/anreact folder, run:

npm test

For jsample (jserv-sample), see Quick Start.

For React Controls, run tests in js/anreact/test/sessionless.
